# PRODIGY_WD_03
A responsive and interactive Tic-Tac-Toe Web Application built using HTML, CSS and Javascript.Tic-Tac-Toe Web Application using HTML, CSS & JavaScript with player turn tracking, win detection, draw checking, and restart functionality.

# Features:
 * Interactive Tic-Tac-Toe gameplay
 *  Two-player mode (X & O)
 * Win condition detection
 * Draw game detection
 * Restart / Reset game option
 * Responsive and user-friendly interface
 * Real-time player turn tracking
  
# Technologies Used:
* HTML – Structure of the web application
* CSS – Styling and responsive design
* JavaScript – Game logic and functionality
